🚀 Fitness Hub - Complete Setup Guide

1️⃣ Firebase Configuration

Step 1: Create Firebase Project

  1. Go to Firebase Console
  2. Click "Add project"
  3. Enter project name: "fitness-hub"
  4. Enable Google Analytics (Optional)
  5. Click "Create project"

Step 2: Get Firebase Config

  1. In Firebase Console, click the gear icon, go to Project Settings
  2. Under "Your apps", select "Web" (or add if not present)
  3. Copy the Firebase configuration
  4. Paste it in assets/js/script.js at line 124

Your config should look like:

const firebaseConfig = {
  apiKey: "YOUR_API_KEY_HERE",
  authDomain: "fitness-hub-xxxxx.firebaseapp.com",
  projectId: "fitness-hub-xxxxx",
  storageBucket: "fitness-hub-xxxxx.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abcdef123456"
};

Step 3: Enable Authentication

  1. Go to Firebase Console > Authentication
  2. Click "Get Started"
  3. Enable "Email/Password" sign-in method
  4. Enable "Google" sign-in (optional)
  5. Save changes

Step 4: Setup Firestore Database

  1. Go to Firebase Console > Firestore Database
  2. Click "Create Database"
  3. Start in "Production Mode"
  4. Select your region (closest to your users)
  5. Click "Enable"

Create collections:

Collection Name Document Format
users uid (auto-generated)
workouts userId + workoutId
diet_plans userId + planId
progress userId + progressId
payments userId + paymentId
💡 Tip: Uncomment the Firebase initialization code in assets/js/script.js and assets/js/auth.js after adding your config.

2️⃣ Razorpay Payment Integration

Step 1: Create Razorpay Account

  1. Go to Razorpay.com
  2. Sign up for a merchant account
  3. Verify your email and phone
  4. Complete KYC verification
  5. Activate your account

Step 2: Get API Keys

  1. In Razorpay Dashboard, go to Settings > API Keys
  2. Copy your "Key ID" (Public Key)
  3. Copy your "Key Secret" (Private Key)
  4. Update in assets/js/auth.js

Update in payment processing function:

const options = {
    key: 'YOUR_RAZORPAY_KEY_ID',
    amount: selectedPlan.price * 100,
    currency: 'INR',
    name: 'Fitness Hub',
    description: `${selectedPlan.name} Membership`,
    prefill: {
        name: userFormData.fullName,
        email: userFormData.email,
        contact: userFormData.phone
    },
    handler: function(response) {
        verifyPayment(response.razorpay_payment_id);
    }
};

const rzp = new Razorpay(options);
rzp.open();
⚠️ Security Note: Never commit your API keys to GitHub. Use environment variables in production.

3️⃣ Stripe Payment Integration (Alternative)

Step 1: Create Stripe Account

  1. Go to Stripe.com
  2. Click "Start now"
  3. Fill in your details
  4. Verify email
  5. Activate account

Step 2: Get Publishable Key

  1. In Stripe Dashboard, go to API Keys
  2. Copy "Publishable key"
  3. Update in assets/js/auth.js
const stripe = Stripe('YOUR_STRIPE_PUBLISHABLE_KEY');

stripe.createPaymentMethod({
    type: 'card',
    card: cardElement,
    billing_details: {
        name: userFormData.fullName,
        email: userFormData.email
    }
}).then(function(result) {
    if (result.paymentMethod) {
        completeStripePayment(result.paymentMethod.id);
    }
});

4️⃣ Email Service Setup

Using Firebase Cloud Functions

  1. Install Firebase CLI: npm install -g firebase-tools
  2. Login: firebase login
  3. Initialize: firebase init functions
  4. Create function to send emails
// functions/index.js
const functions = require("firebase-functions");
const nodemailer = require("nodemailer");

const transporter = nodemailer.createTransport({
    service: "gmail",
    auth: {
        user: process.env.GMAIL_EMAIL,
        pass: process.env.GMAIL_PASSWORD
    }
});

exports.sendWelcomeEmail = functions.auth.user().onCreate(async (user) => {
    const mailOptions = {
        from: "noreply@fitnesshub.com",
        to: user.email,
        subject: "Welcome to Fitness Hub!",
        html: `Welcome ${user.email}! Your account is ready.`
    };
    
    return transporter.sendMail(mailOptions);
});

6️⃣ Admin Dashboard Setup

Create Admin Credentials

Create an admin user in Firebase with special privileges:

  1. Go to Firebase > Authentication
  2. Create user with email: admin@fitnesshub.com
  3. Set custom claims to mark as admin
// In Firebase Cloud Functions or Admin SDK
admin.auth().setCustomUserClaims('uid_here', {
    admin: true,
    created: new Date()
}).then(() => {
    console.log('Admin claims set');
});

8️⃣ Deployment

Deploy on Firebase Hosting

  1. Install Firebase CLI: npm install -g firebase-tools
  2. Login: firebase login
  3. Initialize: firebase init hosting
  4. Select your project
  5. Set public directory to ./
  6. Deploy: firebase deploy

Your app will be live at: https://your-project.firebaseapp.com

Deploy on Vercel (Alternative)

  1. Push to GitHub
  2. Connect to Vercel
  3. Select repository
  4. Deploy automatically

🔧 Troubleshooting

Common Issues

Issue: "Firebase is not defined"

Solution: Add Firebase CDN in HTML head

<script src="https://www.gstatic.com/firebasejs/9.0.0/firebase-app.js"></script>

Issue: "CORS error with payment gateway"

Solution: Configure CORS in Firebase settings

Issue: "Payment not processing"

Solution: Check Razorpay/Stripe keys, verify SSL certificate

Issue: "Data not saving to Firestore"

Solution: Check Firebase Security Rules, enable write access

# 🏋️ FITNESS HUB - PREMIUM GYM & FITNESS MANAGEMENT

![Fitness Hub Hero](./assets/images/hero.png)

## 🌟 Modern, High-End Fitness Solution
Fitness Hub is a state-of-the-art gym management system designed for a premium user experience. It features a sleek dark-mode dashboard, membership management, and real-time analytics for administrators.

### 🚀 Key Features
- **Modern UI/UX**: Designed with vibrant colors, smooth transitions, and premium aesthetics.
- **User Dashboard**: Personalized experience with workout plans (mock), diet plans (mock), and billing history.
- **Membership Management**: Tiered membership plans with an integrated (simulated) payment flow.
- **Admin Analytics**: Centralized hub for monitoring users, revenue, and membership trends.
- **Robust Authentication**: Optimized login/logout flow with secure session management.

### 🛠️ Technology Stack
- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6+), Bootstrap 5.
- **Backend**: Node.js, Express.
- **Database**: Supabase / MongoDB.
- **Libraries**: Chart.js for data visualization.

### 📦 Installation & Setup
1. **Clone the repository:**
   ```bash
   git clone https://github.com/himanshu99-gethub/fitness_hub.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the server:**
   ```bash
   node server-supabase.js
   ```
